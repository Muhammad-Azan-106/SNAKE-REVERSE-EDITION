# SNAKE-REVERSE-EDITION
A unique web-based twist on the classic Snake game developed using JavaScript and HTML5 Canvas. It features custom "Wall Wrapping" logic and a "Reverse Flip" mechanic where the snake's body array is dynamically reversed in real-time based on user input.
